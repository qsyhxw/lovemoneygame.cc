Common Issues & Troubleshooting

Windows: "Windows protected your PC" warning ‚ñº

Cause: Windows SmartScreen blocks unsigned applications from unknown publishers.

Solution:

  1. Click "More info" on the warning dialog
  2. Click "Run anyway" button that appears
  3. The game will launch and this warning won't appear again for this file

Note: This is safe if you downloaded from the official itch.io page. Independent games are rarely digitally signed due to cost.

macOS: "Cannot be opened" or "Moved to trash" error ‚ñº

Cause: macOS Gatekeeper prevents unsigned apps from running by default.

Solution Method 1 (Recommended):

  1. Open System Settings ‚Üí Privacy & Security
  2. Scroll to the Security section
  3. Look for a message about LoveMoney being blocked
  4. Click "Open Anyway"
  5. Confirm in the popup dialog

Solution Method 2 (Alternative):

  1. Right-click (or Control-click) the app icon
  2. Select "Open" from the menu
  3. Click "Open" in the security dialog that appears
Cannot find the executable file after extracting ‚ñº

Windows: Look for a file with the game's icon (often named "LoveMoney.exe" or similar). It should be in the main folder after extraction.

macOS: Look for a file with the .app extension (it will appear as an application icon, not a folder).

Tip: Make sure you extracted the archive completely. Don't try to run the game directly from inside the .zip file.

Game crashes on launch or won't start ‚ñº

Try these steps:

  1. Verify you downloaded the correct version for your operating system
  2. Re-extract the archive to a different location (avoid paths with special characters)
  3. Temporarily disable your antivirus and try again
  4. Update your graphics drivers
  5. If using Windows, try running the .exe as administrator (right-click ‚Üí Run as administrator)
English version has Russian menu/settings ‚ñº

Known Issue: According to player reports, the English version may still show some menu elements in Russian.

Workaround: The main story text is in English. For menu navigation, you can take a screenshot of the Russian menu and use an online translator (like Google Translate's image translation feature) to understand the options.
